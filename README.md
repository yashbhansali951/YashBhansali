# Yash Bhansali - Portfolio Website

A minimalistic, modern portfolio website showcasing my professional experience, education, and skills.

## Features

- **Minimalistic Design**: Clean and professional layout with subtle animations
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: Subtle fade-in and scroll animations for better user experience
- **Modern UI**: Contemporary design with a minimal color scheme

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (Vanilla JS)
- Google Fonts (Inter)

## Getting Started

### Local Development

1. Clone this repository:
```bash
git clone https://github.com/yourusername/YashBhansali_Resume.git
cd YashBhansali_Resume
```

2. Open `index.html` in your web browser, or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js (http-server)
npx http-server
```

3. Navigate to `http://localhost:8000` in your browser

## GitHub Pages Deployment

To host this website on GitHub Pages:

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" in the left sidebar
4. Under "Source", select the branch (usually `main` or `master`)
5. Select the folder (usually `/root`)
6. Click "Save"
7. Your site will be available at `https://yourusername.github.io/YashBhansali_Resume/`

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #2c2c2c;
    --accent-color: #4a90e2;
    /* ... */
}
```

### Content
Edit the content directly in `index.html` to update your information.
